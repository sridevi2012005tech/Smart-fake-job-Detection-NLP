# Smart Fake Job Detection using NLP & Machine Learning

## 📌 Problem Statement
Online job portals have thousands of fraudulent job postings that scam fresh graduates. This project uses AI to automatically classify job posts as **Real** or **Fake** to protect job seekers.

## 🛠️ Tech Stack
- **Language**: Python 3.10
- **NLP**: NLTK, Text Preprocessing, TF-IDF Vectorization
- **ML Algorithms**: Logistic Regression, Random Forest, SVM
- **Libraries**: Pandas, Scikit-learn, Matplotlib, Seaborn
- **Tools**: Git, GitHub

## 🚀 Key Features
- Cleaned and preprocessed 17,000+ job descriptions using NLP techniques
- Implemented TF-IDF for feature extraction from text data
- Achieved **92% Accuracy** and **89% Precision** with Random Forest Classifier
- Model predicts fake jobs with confidence score

## 📊 Results
| Model | Accuracy | Precision |
| --- | --- | --- |
| Random Forest | 92% | 89% |
| Logistic Regression | 88% | 85% |

## ⚙️ How to Run
1. Clone: `git clone https://github.com/sridevi2012005tech/Smart-fake-job-Detection-NLP`
2. Install: `pip install -r requirements.txt`
3. Run: `python main.py`
